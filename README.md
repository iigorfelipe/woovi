## Tecnologias Utilizadas

- TypeScript
- React - `Vite (react-ts)`
- Tailwindcss
- Zod

## Executando o Projeto Localmente

1. Clone o repositório para sua máquina:
    ```bash
    git clone git@github.com:iigorfelipe/woovi.git
    ```

2. Entre na pasta do projeto:
    ```bash
    cd woovi
    ```

3. Instale as dependências:
    ```bash
    npm install
    ```

4. Execute o projeto:
    ```bash
    npm run dev
    ```
